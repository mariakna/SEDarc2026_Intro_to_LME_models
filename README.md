# A Gentle and Practical Introduction to Linear Mixed-Effects models

This repository contains all materials used for the *A Gentle and Practical Introduction to Linear Mixed-Effects Models* class, delivered as part of the specialist skills training offered by SEDarc DTP.

The session provides an accessible introduction to fitting linear mixed-effects models in R, combining key theoretical concepts with practical, hands-on examples. Working with a real dataset, you will learn how to prepare data for analysis, compute descriptive statistics, visualise data, specify contrast coding, transform variables, determine an appropriate random-effects structure, fit mixed-effects models, and conduct model diagnostics. 
The session also introduces generalised linear mixed-effects models as an alternative approach for analysing data without transforming the dependent variable.

In addition to these practical skills, the course explains why mixed-effects models are needed, how they relate to traditional statistical approaches such as t-tests, ANOVAs, and linear regression, and when they should be used in research.

The class materials are contained in the HTML file (and its source R Markdown file). 
The repository also includes the dataset used in the class (*pwiExp_German.csv*), an R script (*geom_flat_violin.R*) required for plotting, and an image (*pwi.png*) required to compile the HTML file correctly.
